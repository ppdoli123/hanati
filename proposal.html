프로젝트 2 : 화장품 추천 서비스
진행 기간 2023.03.06 ~ 2023.11.23 주요 기능
1. 감성 분석
-약 110,000건의 리뷰 데이터로 긍정, 부정 키워드 추출
-제품의 긍정, 부정 비율을 측정
2. 개인화 추천 서비스 제공
-제품 & 키워드별 검색 제공
-지성, 건성 등 피부타입에 해당 되는 제품 추천
-사용자 즐겨찾기 구현
3. 상세 페이지
-가장 많은 키워드를 바탕으로 Hot Topic 제공
-제품 가격, 사진, 리뷰어 특징, 제품 리뷰 정보 확인 가능
-상품 클릭 시 해당 제품 사이트로 이동
-해당 제품과 가장 비슷한 키워드를 가진 제품 추천
활동 기관 OO대학교 캡스톤 대회
참여 인력 5명
수행 역할 팀장 및 Back-End 담당
수행 내용
및 기여도
-피부 타입에 맞는 추천 서비스 기획(Figma) : 80%
-데이터베이스 구축(Firebase) : 100%
-메인 화면 기능, 상세 페이지 기능 구현 (Android Studio) : 100%
-형상 관리(Git) : 50%
목적 및
필요성
-기존 쇼핑몰 앱 내 타입별 추천 기능 부족으로 인한 사용자 맞춤형 제품 제안이 어려움
-화장품 정보를 검색할 떄 자신의 피부 타입에 맞는 제품을 추천하여 사용자가 직접 제품을 찾는 수고를 줄일 수 있음
