웹 개발에 관심이 많아, 프론트엔드 개발에 필요한 여러 기술을 스스로 학습하고 있습니다. 제가 주로 사용하는 프론트엔드 기술은 HTML, CSS, JavaScript입니다. 이를 기반으로 최근에는 React.js 프레임워크를 배우기 시작했습니다. React를 통해 컴포넌트 기반의 개발 방식과 상태 관리의 중요성을 이해하게 되었고, 이를 실제 프로젝트에 적용해 보았습니다. 또한, Redux를 사용하여 복잡한 상태 관리를 경험했으며, 이 과정에서 불변성을 유지하는 방법과 효율적인 데이터 흐름을 설계하는 방법을 배웠습니다. 프로젝트를 진행하면서 CSS 프레임워크인 Bootstrap과 Material-UI를 사용해 보았습니다. 이 도구들은 반응형 디자인을 손쉽게 구현할 수 있게 해주어 사용자 경험을 크게 향상시키는 데 기여했습니다. 이러한 경험을 통해 사용자 중심의 UI/UX 디자인의 중요성을 더 깊이 이해하게 되었습니다.
